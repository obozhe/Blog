# Blog app built with react.js, node.js and MongoDB

# Installation instruction

## Run server

0. Go to server folder.

```bash
cd blog-server
```

1. Install.

```bash
npm install
```

2. Run.

```bash
npm start
```

## Run client

0. Go to client folder.

```bash
cd blog-client
```

1. Install.

```bash
npm install
```

2. Run.

```bash
npm start
```

3. Open app in browser at localhost:3000
